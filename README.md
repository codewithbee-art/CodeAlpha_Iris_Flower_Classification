# Iris Flower Classification using Machine Learning

## Objective
The objective of this project is to build a machine learning model to classify Iris flower species (Setosa, Versicolor, Virginica) based on their petal and sepal measurements.

---

## Dataset
This project uses the built-in Iris dataset available in Scikit-learn.  
The dataset contains:
- 150 samples
- 4 features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 3 target classes (species)

---

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Workflow
The project follows a standard machine learning pipeline:

1. Data Loading (using Scikit-learn dataset)
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Train-Test Split
5. Model Training
6. Model Evaluation
7. Model Comparison

---

## Models Used
Three machine learning models were trained and compared:

- Logistic Regression
- Random Forest Classifier
- Support Vector Machine (SVM)

Each model was trained on the training dataset and evaluated on the test dataset.

---

## Evaluation
Model performance was evaluated using:

- Accuracy Score

Each model's accuracy was calculated and compared to determine the best-performing model.

---

## Result
All models performed well on the dataset.  
Random Forest and SVM generally provided higher accuracy compared to Logistic Regression, showing strong capability in handling classification tasks.

---

## Conclusion
This project demonstrates how machine learning can be used for classification problems.  
By comparing multiple models, we can identify the most suitable algorithm for the dataset and improve prediction performance.
